Security

Security is AnyFetch top priority.

All files are stored in secure indices, ensuring a strict no-access rule for all data.
All server traffic is encrypted using SSL.

A chaos-monkey script runs continuously trying to breach our system, ensuring your data-policies are correctly applied.

Finally, the original document is never stored on a hard drive: it is only kept in RAM for the analysis. You remain the sole owner of the original document.
